# 用户意图

## 目标用户
本文档主要面向以下两类用户：
1. **开发者**：希望使用 AIGNE CLI 工具进行 Agent 开发、测试和部署的开发人员
2. **AIGNE 框架使用者**：需要通过命令行工具管理 AIGNE 应用、连接 AIGNE Hub、运行 MCP 服务的用户

## 使用场景
- 初次接触 AIGNE CLI 时，需要了解工具的安装、基本功能和使用方法
- 开发 AIGNE 应用时，需要查阅具体命令的用法和参数说明
- 部署和管理 AIGNE 应用时，需要参考部署、MCP 服务和 Hub 连接相关的操作指南
- 运行测试和评估时，需要了解测试命令和评估工具的使用方法
- 排查问题时，需要查阅命令选项、日志配置等参考信息

## 文档侧重点
本文档采用**使用指南 + 命令参考**的组合形式：
- 提供清晰的安装和快速开始指南
- 详细说明每个命令的功能、参数和使用示例
- 提供实际场景的操作指引（如创建项目、运行 Agent、部署应用、连接 Hub）
- 提供配置说明和常见问题解答

不需要深入的架构设计或源码分析，重点是命令行工具的实际使用。
