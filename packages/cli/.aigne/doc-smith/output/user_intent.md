# 用户意图

## 目标用户
本文档主要面向使用 AIGNE Framework 进行 AI agent 开发的开发者。他们需要了解如何使用 AIGNE CLI 工具来创建、运行、测试和部署 AI agents。

## 使用场景
- 开发者首次接触 AIGNE Framework 时，需要快速了解 CLI 工具的核心功能和如何开始
- 开发过程中需要查阅具体命令的使用方法、参数说明和选项
- 需要了解如何连接到 AIGNE Hub、管理凭证、使用远程 agents
- 部署和测试 agents 时需要参考配置和最佳实践
- 评估 agent 性能和调试问题时需要使用可观测性工具

## 文档侧重点
本文档采用**命令参考手册**为主的形式：
- 提供完整的命令参考文档，详细说明每个命令的用法、参数、选项和示例
- 仅提供两个最常见的使用指南（快速开始和基本工作流程）
- 每个命令都有清晰的语法说明、参数列表和实际使用示例
- 命令按功能分类组织，便于查阅

不需要过多的教程内容、框架内部实现细节和架构设计说明，重点在于作为命令速查手册使用。
