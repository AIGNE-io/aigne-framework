# Planning

项目规划与演进记录。

## 目录结构

```
planning/
├── ideas/      # 想法收集，未成型的概念
├── intents/    # 明确的意图规格，准备实现
├── roadmap/    # 路线图与里程碑
└── changelog/  # 重大升级与变更记录
```

## 文件命名约定

- Ideas: `YYYY-MM-DD-<slug>.md`
- Intents: `<feature-name>.md`
- Roadmap: `YYYY-Q<N>.md` 或 `v<version>.md`
- Changelog: `YYYY-MM-DD-<title>.md`
