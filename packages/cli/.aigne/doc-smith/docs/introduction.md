# Introduction

`@aigne/cli` is the official command-line tool for the **AIGNE Framework**, designed to simplify the development, testing, and deployment processes for AIGNE applications.

## Key Features

- **Project Creation**: Quickly create new AIGNE projects with predefined file structures and configurations using `aigne create`.
- **Agent Running**: Easily run and test AIGNE agents locally with `aigne run`.
- **Testing Support**: Built-in test command for unit and integration testing via `aigne test`.
- **MCP Services**: Launch agents as [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) servers using `aigne serve-mcp`.
- **Observability**: Monitor agent activities and performance with `aigne observe`.
- **Multi-model Support**: Seamlessly work with OpenAI, Claude, XAI, and other model providers.

## Installation

Install the CLI globally using your preferred package manager:

### npm
```bash
npm install -g @aigne/cli
```

### yarn
```bash
yarn global add @aigne/cli
```

### pnpm
```bash
pnpm add -g @aigne/cli
```
