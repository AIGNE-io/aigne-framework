# 用户指南

欢迎阅读 AIGNE 用户指南。本指南专为非技术背景的读者设计，包括企业领导者、管理人员以及任何对如何利用 AI 自动化复杂任务和简化业务流程感兴趣的人士。您无需任何编程知识即可轻松理解本指南。

我们的目标是以简单易懂的方式解释 AI Agent 和工作流的核心概念。您将了解什么是 AI Agent，它们如何独立或以团队形式工作，以及如何编排它们来解决实际问题，最终推动业务成果。
