# Command Reference

The `@aigne/cli` provides a suite of commands to manage the entire lifecycle of your AI agents.

## Usage Syntax

```bash
aigne <command> [options]
```

## Available Commands

| Command | Description |
| :--- | :--- |
| [`create`](/commands/create.md) | Create a new AIGNE project with agent config files. |
| [`run`](/commands/run.md) | Run AIGNE agents locally for development and debugging. |
| [`test`](/commands/test.md) | Run tests in the specified agents directory. |
| [`serve-mcp`](/commands/serve-mcp.md) | Serve agents as an MCP server. |
| [`observe`](/commands/observe.md) | Start the observability service. |
| [`deploy`](/commands/deploy.md) | Deploy agents to a remote environment. |
| [`eval`](/commands/eval.md) | Evaluate agent performance. |
| [`hub`](/commands/hub.md) | Interact with the AIGNE Hub. |
| [`app`](/commands/app.md) | Manage AIGNE applications. |

## Global Options

- `--help`, `-h`: Show help information for a command.
- `--version`, `-v`: Show the CLI version number.
